# Prerequirsite
pipenv:latest

# how to run :

1. Clone this repo `git clone https://github.com/hardik-dadhich/MediOne`
2. Go to project root directory and run `pipenv shell`
3. Run `pipenv install`
4. Then finally run `python manage.py runserver`
